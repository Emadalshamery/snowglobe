Snowglobe is a gem that helps erect and destroy Rails applications for use in
tests.

## Installation

Add this line to your project's Gemfile:

``` ruby
gem 'snowglobe'
```

And then execute:

``` bash
bundle
```

Alternatively, install it yourself as:

``` bash
gem install snowglobe
```
